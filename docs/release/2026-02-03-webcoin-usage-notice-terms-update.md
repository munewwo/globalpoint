# ウェブコインの利用に関する注意事項（利用規約改定のお知らせ）

平素より当社サービスをご利用いただき、誠にありがとうございます。

昨年末、警視庁より、風俗営業許可を受けて営業する都内のアミューズメントカジノ店舗に対して注意喚起が行われたことを受け、当社ではウェブコインの適正な利用について改めて検討いたしました。

これを踏まえ、リングゲームにおいてウェブコインを利用する場合には、料金システム等を含む運用内容について当社の事前承認が必要となること等、新たな注意事項を追加した利用規約へ改定いたしますので、お知らせいたします。

なお、トーナメントにおけるウェブコインの利用については、従来どおりご利用いただけます（本改定による変更はありません）。

つきましては、現在リングゲームでウェブコインをご利用の店舗におかれましては、【提供停止期限：2月28日】までに、既存の運用（いわゆるウェブコインリング）による提供を停止くださいますようお願い申し上げます。

また、【運用開始日：3月1日】以降もリングゲームでのウェブコイン利用の継続を希望される事業者様は、【申請期限：2月20日】までに所定の申請手続きを行っていただく必要があります。

申請方法および運用内容の詳細につきましては、改めてご案内いたします。

店舗運営事業者の皆様におかれましては、ウェブコインの適正な利用についてご理解いただき、業界の健全かつ継続的な発展と社会的信頼の構築に向けた取り組みに、引き続きご理解とご協力を賜りますようお願い申し上げます。
