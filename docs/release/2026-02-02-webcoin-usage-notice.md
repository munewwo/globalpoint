# ウェブコインの利用に関する注意事項

ウェブコインのご利用にあたっては、下記を参考として、風営法の遵守および当社の利用規約に沿ってご利用いただくようお願いいたします。当社が適切に利用していないと判断したきは、サービスを停止させていただく場合があります。

店舗運営事業者の皆さまには、ウェブコインの適正な利用についてご理解いただき、業界の健全かつ継続的な発展と社会的信頼の構築に向けた取り組みについて、ご理解とご協力をお願いいたします。

## 店舗が提供する賞品・景品ではありません

ウェブコインは、グローバルポイント社（第三者）とプレイヤーとの間で任意に締結される選手契約（業務委託契約）に基づき、委託料として付与される有償ポイントです。

店舗からプレイヤーへ付与するものではありません。

契約プレイヤーは、ウェブコインを使って国内外のトーナメント出場支援が受けられます。

### 風営法の禁止事項

遊技結果によって獲得等したチップ、メダル等を賞品（店内で販売する飲食物も含む）に交換*すること

*店内（他店を含む）の有料サービス等で利用可能な「ポイント」に交換することも含む

### 利用規約の禁止事項

ウェブコインの利用にかかわらず、風営法店がプライズとして賞金を提供することは禁止

ウェブコインは、18歳以上の人のみご利用いただけます。

未成年者からのウェブコインの受付や付与は行わないでください。

## 遊技（預かり）チップはウェブコインに交換しません

ウェブコインは、遊技チップまたは預かりチップを交換したものではありません。グローバルポイント社が定める評価制度と選手契約の上で付与されるポイントです。

リングゲームでは、チップ＝評価とならような評価制度に限って利用を認めます。

会員アプリ「gameid」では、遊技チップまたは預かりチップは表示されません。

### 風営法の禁止事項

遊技の結果によって獲得等したチップ、メダル等を営業所外に持ち出させること

それを客のために保管したことを表示する書面を客に発行すること

### 事業者連盟の方針

「獲得したチップ＝ポイント」という構造とならないこと

### 利用規約の禁止事項

リングゲームでウェブコインを利用するときは、店舗の料金システムを含めた評価制度をグローバルポイント社が定めた上で、認定証を店舗に掲示してください

当社以外が発行したポイントや評価制度に基づいてウェブコインを付与することは禁止させていただきます。

## 著しく射幸心をそそるおそれのあるサービス提供と広告抑止

日本ポーカー事業者連盟の方針と当社の利用規約に基づき、著しく射幸心をそそるおそれのある広告表現は制限する必要があります。

これらに違反した可能性があるとき、注意の上、ウェブコインの利用取り扱いを停止するなどの措置をとらせていただきます。

### 事業者連盟の方針

著しく射幸心をそそるおそれのある広告表現は制限

### 利用規約の禁止事項

ウェブコインの利用にかかわらず、「等価」や「還元率」などを用いて、著しく射幸心をそそるおそれのある広告表現は規制させていただきます

ウェブコインの利用にかかわらず、トーナメントは10万円（税別）、リングゲームは5万円（税別）を超えるサービスの提供は行わないでください

ウェブコインの利用にかかわらず、料金は先払いを推奨とし、後払い額が10万円を超えるときは一度精算してください。
