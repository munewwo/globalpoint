# リリース情報

グローバルポイント社が発表したさまざまなリリースを保管するフォルダです。

## 運用ルール

- **対象**: 対外的に発表・公開した内容（告知、注意事項、見解、ガイドライン、ロードマップ等）を「リリースノート」として保管します。
- **命名**: `YYYY-MM-DD-<内容が分かる短い英字スラッグ>.md` を推奨します。
- **追加手順**: 新しいリリースノートを追加したら、この `README.md` の一覧表にも追記し、**最新日付を上**に並べます。
