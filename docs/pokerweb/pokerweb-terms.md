# POKERWEB 利用規約

本利用規約（以下「本規約」といいます。）は、合同会社グローバルポイント（以下「当社」といいます。）が、PW SOFTWARES DE GESTAO LTDAから日本国内における運用業務等の委託を受け、登録事業者に対して提供する役務の条件、ならびに登録事業者が日本国内でポーカールーム向けシステム「POKERWEB」（以下「本サービス」といいます。）を利用するにあたり当社との間で遵守すべき事項を定めるものです。

本サービス（POKERWEB）自体はPW SOFTWARES DE GESTAO LTDAが提供するものであり、通常、本サービスの利用条件はPW SOFTWARES DE GESTAO LTDAが定めるところによります。登録事業者は、PW SOFTWARES DE GESTAO LTDAが別途定める利用条件（規約・ポリシー等）が適用される場合があることを理解したうえで、日本国内で本サービスを利用するにあたり、本規約および個別規程にも同意するものとします。

本サービスには、当社が別途定める条件により提供されるオプション機能（例：ウェブコイン）が含まれる場合があります。オプション機能に関する個別の条件は、本規約のほか、当社が別途定める別紙・規程・申込書等（以下総称して「個別規程」といいます。）により定めるものとし、個別規程と本規約の内容が異なる場合は、個別規程が優先して適用されます。

## 第1条（適用）

1. 本規約は、本サービスの提供条件及び本サービスの利用に関する当社と登録事業者との間の権利義務関係を定めることを目的とし、登録事業者と当社との間の本サービスの利用に関わる一切の関係に適用されます。

2. 当社が当社ウェブサイト上で掲載する本サービス利用に関するルール（https://pokerguild.jp/pokerweb/policy/）は、本規約の一部を構成するものとします。

3. 本規約の内容と、前項のルールその他の本規約外における本サービスの説明等とが異なる場合は、本規約の規定が優先して適用されるものとします。但し、個別規程に別段の定めがある場合は、当該個別規程が優先して適用されるものとします。
