## 概要

**ポーカーを、もっと便利に楽しもう。**

POKERWEBは、ブラジル・PWソフトウェア社が開発するポーカールーム向けのオペレーション＆マネジメントシステムです。

- プレイヤーアプリ **GameID** と連動した顧客管理
- トーナメント・リングゲームのオペレーション
- アワーディング、ランキングなどポーカールームに必要な機能を一括提供
- **AWS（アマゾンウェブサービス）** 上で運用され、海外のポーカールームにも耐えうるセキュリティを採用

日本公式代理店である**ポーカーギルド社**は、日本向け機能の開発に加え、日本のポーカールーム向けにシステム導入のご案内を行っています。

### 主なメリット

| 項目 | 内容 |
|------|------|
| **コインで集客する** | 加盟店共通で利用できるポイントサービス。大型大会でも使用可能で、集客・売上アップが期待できる |
| **アプリで顧客登録** | GameIDの登録情報をポーカールームとシェアする形で会員登録。コイン受取／使用、JOPTなどのチケット発行・使用にも対応 |
| **必要機能を一本化** | レジアプリ、トーナメントクロック、チケットシステム、ポイントシステムなどを POKERWEB で一括管理 |

## 申し込み手順

POKERWEBの導入を検討される方は、**サポートデスク公式LINE**までお問い合わせください。

### 1. ご案内

- 基本的な機能はヘルプページで確認可能
- ご希望に応じてオンラインでご相談を実施
- 現在のシステムや提供サービス内容を伺い、最適な導入プランをご案内

### 2. プランの選択

利用内容によりプランと料金が決まります。詳細は**利用プランと機能一覧**をご覧ください。

### 3. POKERWEB COINについて

コインシステムは POKERWEB の**オプションサービス**で、別途広告費が必要です。  
コイン利用をご希望の店舗様は、クラブ作成完了後にサポートデスクにご相談ください。

### 4. オンラインサポート

- 運用システムの提供に加え、実際のポーカールームオペレーションを含めたご提案を実施
- **Google Meet** によるオンラインサポートあり。ご希望の方は予約ページから申し込み

### 5. ハードウェアの準備

- 既存のPC・店舗備え付けモニタ（トーナメントクロック用）・配線などは可能な限り活用可能
- 確認後、導入に必要なハードウェアの見積りを発行  
- 詳細: [ハードウェアの設置例](https://pokerguild.jp/pokerweb/hardware_ex/)

### 6. 店舗ロゴの準備

- GameIDアプリ上に表示する店舗ロゴを設定可能
- システム作成時、または作成後にお申し付けください

**ロゴの条件:**

- **背景透過のPNG画像**を推奨（JPEGや背景付きは表示が乱れる場合あり）
- 細部の凝った画像やアプリ背景と同系色のロゴは視認しづらい場合あり
- **画像の提出はギガファイル便でお願い**（LINE経由だと圧縮でPNGがJPEGになる場合があります）
