# 登録事業者向け共通規程

本規程は、合同会社グローバルポイント（当社）が取り扱うサービス（POKERWEB、GameID、ウェブコインその他当社が別途定めるサービスを含みますが、これらに限りません。）に共通して適用される、登録事業者の順守事項、苦情対応、調査協力、情報漏えい時の対応、掲示・表示および契約終了時の措置に関する基本方針を定めるものです。

各サービスの利用規約・個別規程に別段の定めがある場合は、当該利用規約・個別規程が優先して適用されます。本規程に定めのない事項については、各サービスの利用規約の定めに従うものとします。

## 第1条（法令遵守・許認可）

1. 登録事業者は、本サービスの利用および登録事業者が行う事業の運営にあたり、関連するすべての法令（風俗営業等の規制及び業務の適正化等に関する法律、資金決済に関する法律、不当景品類及び不当表示防止法、消費者契約法、特定商取引に関する法律、個人情報の保護に関する法律その他適用のある法令を含みますが、これらに限りません。）および監督官庁の指導等を遵守するものとします。
2. 登録事業者は、事業の運営に必要な許認可・届出・登録等を、自己の責任と費用において取得・維持しなければなりません。
3. 登録事業者は、当社から要求を受けた場合、許認可等の取得状況その他法令遵守に関する資料を速やかに当社に提出するものとします。

## 第2条（正確な情報の提供・広告表示）

1. 登録事業者は、当社に提出する資料または当社に届け出る登録事項その他の情報が、正確かつ最新の内容であることを表明し、保証するものとします。
2. 登録事業者は、本サービスに関連する広告、宣伝、販売促進その他の表示（店舗内外の掲示物、ウェブサイト、SNS等を含みますが、これらに限りません。）を行うにあたり、以下の事項を遵守するものとします。
   1. 不当景品類及び不当表示防止法、消費者契約法その他の関係法令を遵守すること
   2. 当社の信用またはイメージを毀損するおそれのある表示を行わないこと
   3. 当社と登録事業者との間に、代理・提携・共同事業その他本サービスの利用関係を超えた取引関係があることを示唆する態様で表示を行わないこと
   4. 当社または当社の従業員が、登録事業者のサービスを推奨または保証しているかのように表示しないこと
   5. その他、当社が不適切と判断する態様で表示を行わないこと
3. 当社は、登録事業者の広告表示が不適切であると判断した場合、登録事業者に対し、当該表示の是正または中止を求めることができるものとし、登録事業者はこれに従うものとします。

## 第3条（苦情・紛争対応）

1. 登録事業者は、登録事業者の店舗、サービスまたは商品等に関して、会員、顧客またはその他の第三者から苦情（サービス内容、料金、付与リクエストに関する苦情を含みますが、これらに限りません。）を受けた場合、すべて自己の責任と費用において対応し、解決を図るものとし、当社に一切の迷惑を及ぼさないものとします。
2. 登録事業者は、前項の苦情を解決するにあたって、相手方の意向を十分尊重して速やかに対応するとともに、その経過を当社に対して報告するものとします。
3. 登録事業者は、苦情の再発防止のために必要な措置を講じなければなりません。
4. 登録事業者が会員その他の顧客との間のサービス利用に関する取引を合意の上で取り消し、または訂正する場合の手続は、登録事業者が自己の責任で行うものとし、当社は、技術的に対応が困難な場合その他当社が合理的に判断する場合、対応を行わないことがあります。

## 第4条（調査協力・資料提出）

1. 登録事業者は、当社が登録事業者に対し、業務内容、本サービスの利用状況、法令遵守状況、苦情の状況その他当社が必要と認めた事項に関して調査、報告または資料の提出を求めた場合、速やかにこれに応じるものとします。
2. 当社は、登録事業者管理のため、必要に応じて登録事業者に対して途上審査（登録後の継続的な審査を含みます。）を行うことがあり、登録事業者はこれを承諾するものとします。
3. 当社は、不正利用が発生し、またはそのおそれがあると判断した場合、登録事業者に対し、関連する情報やログの提供その他必要な協力を求めることができるものとし、登録事業者はこれに応じるものとします。

## 第5条（情報漏えい等の事故対応）

1. 登録事業者は、登録事業者が保有する本サービスに関する情報（共有情報、会員に関する情報、アクセス権限に関する情報その他本サービスの利用に関連して取得した情報を含みます。）の滅失、毀損または漏えい（以下「漏えい等」といいます。）が生じた場合、速やかに当社に対し、漏えい等の発生の日時、内容その他の詳細事項について報告をしなければなりません。
2. 登録事業者は、前項の漏えい等が生じた場合、速やかに漏えい等の原因を究明するために必要な調査を行い、当社に報告するものとします。また、この場合、登録事業者は、漏えい等の再発防止のために必要な措置を講じたうえで、その内容を当社に書面で報告しなければなりません。
3. 当社は、登録事業者が実施した前項の調査または再発防止のための措置が不十分であると認めた場合、その他当社が必要と認める場合は、登録事業者に対し、追加調査、改善の要求その他必要な措置または指導を行うことができるものとし、登録事業者はこれに従うものとします。ただし、当社による指導は、登録事業者を免責するものではありません。

## 第6条（店舗情報の掲示・表示）

1. 登録事業者は、本サービスの利用開始後、当社が定める方法に従い、GameIDアプリ上またはPOKERWEB上に店舗情報（店舗名称、所在地、連絡先、ロゴその他当社が指定する情報）を登録することがあります。
2. 登録事業者は、前項により登録した店舗情報に関し、当社または当社が指定する者のサービス（アプリ、ウェブサイト、販促資料等を含みます。）に掲載される場合があること、また、当社の判断で掲載を変更または停止する場合があることを、あらかじめ承諾します。
3. 登録事業者は、当社の商号、商標、ロゴその他の表示を使用する場合、当社が別途定める条件に従うものとし、当社の事前の承諾なく、本サービスの利用関係を超えた態様で使用してはなりません。
4. 登録事業者は、店舗情報の内容が常に正確かつ最新であるよう、定期的に更新を行うものとします。

## 第7条（契約終了時の措置）

1. サービス利用契約が終了した場合（退会、登録抹消、解除その他理由の如何を問いません。）、登録事業者は、速やかに以下の措置を講じなければなりません。
   1. 本サービスの利用中であることを示す掲示物、案内、標章その他の表示を撤去または廃棄すること
   2. 当社の商号、商標、ロゴその他当社の表示の使用を中止すること
   3. 当社から提供を受けた販促資料、マニュアルその他の物品がある場合は、当社の指示に従い返却または廃棄すること
2. 前項の措置が完了しないことにより登録事業者または第三者に損害が生じた場合でも、当社は責任を負いません。

## 第8条（損害賠償）

登録事業者が本規程その他当社が定める利用規約・個別規程に違反し、当社に損害が発生した場合、登録事業者は当該損害を賠償する責任を負います。
