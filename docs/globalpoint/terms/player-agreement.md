# 選手契約（業務委託契約）

合同会社グローバルポイント（以下「当社」といいます。）と、当社が認定したポーカー大会において優れた成績を収めた選手（以下「選手」といいます。）は、以下のとおり業務委託契約（以下「本契約」といいます。）を締結します。

## 第1条（目的）

本契約は、当社が認定したポーカー大会において、優れた成績を収めた選手を当社の認定選手とし、選手のポーカー活動を支援することにより、ポーカー競技の発展および健全な普及を図ることを目的とする。

## 第2条（業務委託等）

1. 当社は、選手に対して、以下に定める業務（以下「本業務」という）を委託し、選手はこれを受託する。

   1. 別途記載された補助対象一覧に記載の履行対象ポーカートーナメント（以下「対象トーナメント」という）に当社の認定選手として参加する業務
   2. 対象トーナメントに参加する際、当社指定のワッペンやステッカーを衣類等に添付して露出する業務
   3. 当社のプロモーション活動や取材、撮影や録画への出演および紹介
   4. 前各号に定める業務に付随する業務
   5. その他、両者で別途合意した業務

2. 当社は、必要に応じ、選手が本業務を行う際に必要となる備品を貸与する。

3. 当社は、本契約期間中、両者協議のうえ、選手に委託する前項の業務の範囲を変更することができる。

## 第3条（委託枠）

1. 選手は、トーナメントで優秀な成績を収めた場合、委託枠を取得するものとする。ただし、別途記載された補助対象一覧の委託料および委託枠が、日本円以外の通貨で記載されている場合、別途当社が定める為替レートに基づき日本円で計算された委託料および委託枠を取得するものとする。

2. 前項の委託枠は、取得から365日間有効とする。

## 第4条（委託料）

1. 当社は、選手に対し、対象トーナメントに選手が参加するにあたり、かかる費用及び別途記載の渡航費用の補助代金を支払う。ただし、それまでに獲得した委託枠の総額（第3条第2項の有効期間を過ぎて無効となった委託枠を除く）からそれまで支払われた委託料の総額を差し引いた金額を上限とする。

2. 委託料の支払いにつき、選手は当社に対し、ポーカー活動にかかる費用の領収書等（以下、「査証」という。）を提出する。なお、査証にて当社が選手のポーカー活動の事実が確認できない場合には、当社は選手に対し、追加の査証の提出を求めることができる。

## 第5条（報告）

当社は、選手に対して、必要に応じ、本業務の状況につき報告を求めることができる。

## 第6条（再委託の禁止）

選手は、当社に事前に通知することなしに、本業務の全部または一部を第三者（以下「再委託先」という）に再委託してはならない。なお、選手の事前の通知の有無にかかわらず、選手による再委託先の使用は、選手の責任において行い、再委託先の責めに帰すべき事由については、すべて選手の責めに帰すべき事由とみなす。

## 第7条（秘密保持）

1. 選手は、本業務の履行過程において当社より受領するあらゆる情報を秘密情報として厳にその機密を保持し、本業務遂行の目的のみに使用する。選手は、本業務遂行のために必要な範囲で弁護士、税理士、公認会計士に開示すべき場合（これらの者にも本条と同じ義務を課すことを前提とする。）を除き、当社の同意なく、第三者に対しかかる秘密情報を開示又は漏洩してはならない。但し、以下のいずれかに該当する情報については、秘密情報に該当しないものとする。

   1. 当社から提供又は開示された時点で、既に公知となっていた情報
   2. 当社から提供又は開示された後、自己の責めによらないで公知となった情報
   3. 当社から提供又は開示された時点で、既に当社に対して秘密保持義務を負うことなく保有していた情報
   4. 法律又は契約に違反することなく第三者から提供又は開示された情報

2. 本契約が終了した場合でも、本条に規定する守秘義務は、本契約から将来に渡り効力を有するものとする。

## 第8条（権利義務の移転禁止）

当社及び選手は、あらかじめ書面により相手方の承諾を得なければ、本契約に定める自己の権利または義務を第三者に譲渡し、または担保に供することができない。

## 第9条（契約の解除）

1. 当社または選手は、他の当事者が次の各号の1つに該当したときは、催告なしに直ちに、本契約の全部または一部を解除することができる。

   1. 本契約に違反し、相当の期間を定めて相手方に対して、その是正を求めたにも関わらず、相手方がその違反を是正しないとき
   2. 相手方の信用、名誉または相互の信頼関係を傷つける行為をしたとき
   3. 破産手続開始、民事再生手続開始、会社更生手続開始、その他倒産手続開始の申立があったとき
   4. 差押え、仮差押え、仮処分、競売の申立、租税滞納処分その他これに準ずる手続があったとき
   5. 支払停止もしくは支払不能に陥ったとき、または、手形または小切手が不渡りとなり、手形交換所より銀行取引停止処分を受けたとき
   6. 合併、解散、清算、事業の全部もしくはその他重要な事業の一部を第三者へ譲渡し、または使用としたとき
   7. その他前各号に類する事情が存するとき

2. 前項に基づく解除は、相手方に対する損害賠償請求を妨げない。

## 第10条（有効期間）

1. 本契約の有効期間は、契約日から365日間とする。ただし、当社が認めるときに限って、期間を延長することができる。

2. 期間満了により、本契約が終了する場合には、両者協議のうえ、本業務に関する清算業務を行う。

3. 当社は、第1項の規定に関わらず、2ヶ月前までに選手に対して書面により通知することにより、本契約を解約することが出来る。

## 第11条（反社会的勢力との取引排除）

1. 選手は、当社が別途定める反社会的勢力の排除に関する規程（共通）（`antisocial.md`）に同意し、これを遵守するものとする。

2. 選手が前項の規程に違反し、または違反するおそれがあると当社が判断した場合、当社は何らの催告を要せず、本契約の全部または一部を解除することができる。

3. 選手は、当社が前項の規程に違反したと認める場合には、通知、催告その他の手続を要しないで、直ちに本契約の全部または一部を解除することができる。

4. 前二項に基づく解除により相手方に損害が生じた場合でも、解除した当事者は賠償の責を負わない。また、当該解除により解除した当事者に損害が生じたときは、違反した当事者はその損害を賠償するものとする。

## 第12条（個人情報の取扱いに関する共通規程）

選手は、本契約の締結にあたり、当社が別途定める以下の共通規程に同意し、これを遵守するものとする。

- 当社プライバシーポリシー（共通）（`privacy.md`）

## 第13条（合意管轄）

この契約に関する紛争については、訴額に応じて東京地方裁判所又は東京簡易裁判所を第1審の専属的合意管轄裁判所とする。

## 第14条（協議）

本契約に定めのない事項及び疑義が生じた事項については、両当事者協議のうえ決定するものとする。
