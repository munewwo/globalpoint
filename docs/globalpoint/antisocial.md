# 反社会的勢力の排除に関する規程（共通）

本規程は、合同会社グローバルポイント（当社）が取り扱うサービス（POKERWEB、GameID、ウェブコインその他当社が別途定めるサービスを含みますが、これらに限りません。）に共通して適用される、反社会的勢力の排除に関する基本方針を定めるものです。

## 1. 定義

本規程において「反社会的勢力」とは、暴力団、暴力団員、暴力団員でなくなった時から5年を経過していない者、暴力団準構成員、暴力団関係企業、総会屋等、社会運動等標ぼうゴロ、特殊知能力暴力集団、その他前各号に準ずる者をいいます。

## 2. 表明・確約

利用者（登録事業者、会員その他当社が別途定める者を含みます。）は、以下を表明し、将来にわたり確約するものとします。

1. 自ら（法人の場合は役員および経営に実質的に関与している者を含みます。）が反社会的勢力に該当しないこと
2. 反社会的勢力と社会的に非難されるべき関係を有しないこと
3. 自らまたは第三者を利用して、暴力的要求行為、法的な責任を超えた不当な要求行為、脅迫的言動・暴力行為、風説の流布・偽計・威力による信用毀損・業務妨害その他これらに準ずる行為を行わないこと

## 3. 解除・利用停止等

当社は、利用者が以下のいずれかに該当すると当社が判断した場合、何らの催告を要せず、当社が別途定める契約の解除、サービス利用停止、登録抹消その他必要な措置を講じることができます。

1. 反社会的勢力に該当することが判明した場合
2. 反社会的勢力と、経営支配・実質関与、不当利用、資金等提供・便宜供与その他社会的に非難されるべき関係を有することが判明した場合
3. 前条の表明・確約に違反した場合

## 4. 再委託先等に関する確約

1. 利用者は、利用者の再委託先その他の取引先（再委託契約が数次にわたるときは、その全てを含みます。以下「再委託先等」といいます。）が、第1条に定める反社会的勢力に該当せず、将来においても該当しないことを確約するものとします。
2. 利用者は、再委託先等が前項に該当することが契約後に判明した場合には、直ちに当該再委託先等との契約を解除し、または契約解除のための措置を執らなければなりません。

## 5. 不当介入への対応

利用者は、利用者または利用者の再委託先等が反社会的勢力から不当要求または業務妨害等の不当介入を受けた場合、これを拒否し、または再委託先等にこれを拒否させるとともに、速やかに当社に報告し、捜査機関への通報および当社の報告に必要な協力を行うものとします。

## 6. 免責等

当社が本規程に基づき措置を講じたことにより利用者に損害が生じた場合でも、当社は法令上必要な場合を除き責任を負いません。また、当社に損害が生じたときは、利用者はその損害を賠償するものとします。
