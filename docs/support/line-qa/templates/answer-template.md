# 回答テンプレ（LINE・加盟クラブ向け）

## 1) 結論（1〜2行）

（結論からお伝えしますと、〜です。）

## 2) 重要ポイント（最大3点）

- （重要点1）
- （重要点2）
- （重要点3）

## 3) 手順（順番が重要な場合のみ）

1. （最初にやること）
2. （次にやること）
3. （最後にやること）

## 4) こちらで確認したいこと（不足情報がある場合のみ）

差し支えなければ、状況確認のために以下を教えてください。

- （例: 店舗名／クラブ名）
- （例: 発生日時）
- （例: 画面のスクリーンショット）
- （例: 該当する取引・イベント・設定の名称）

※個人情報や機微情報が含まれる場合は、社内確認のうえ依頼します。

## 5) 次のサポート提案（必要そうな場合のみ）

もしご希望でしたら、こちらで（例: 設定確認／運用の整理／注意点の共有）までご案内します。

## 6) 参照先（根拠が必要な場合のみ、1〜2件）

- （例）`docs/release/README.md`（リリース一覧）
- （例）`docs/pokerweb/pokerweb-webcoin.md`（ウェブコインの説明）

## 言い回しの注意（法的断定を避ける）

- 「法的に問題ありません」「合法です」などは避ける
- 代わりに「当社の現時点の運用（判断）としては、〜の整理です」「関連する注意事項として〜です」を使う
