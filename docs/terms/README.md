# 用語集（顧客対応・文書作成用）

このファイルは、顧客対応で使用する用語の **正規表記**・**定義**・**注意点（誤解しやすい点）** を集約し、文書間の表記ゆれや未定義語を減らすためのものです。

## 運用ルール

- **追加順**: 新しい言い回し・略語・固有名詞が出てきたら、まずここに追記してから各文書へ反映します。
- **正規表記**: ここで定めた表記を優先し、別表記は「別表記／表記ゆれ」に寄せます。
- **定義の優先順位**: 規約・ポリシー等で文書内定義がある場合は、**その文書の定義が最優先**です（例: 「当社」）。
