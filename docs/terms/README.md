# 用語集（顧客対応・文書作成用）

このファイルは、顧客対応で使用する用語の **正規表記**・**定義**・**注意点（誤解しやすい点）** を集約し、文書間の表記ゆれや未定義語を減らすためのものです。

## 運用ルール

- **追加順**: 新しい言い回し・略語・固有名詞が出てきたら、まずここに追記してから各文書へ反映します。
- **正規表記**: ここで定めた表記を優先し、別表記は「別表記／表記ゆれ」に寄せます。
- **定義の優先順位**: 規約・ポリシー等で文書内定義がある場合は、**その文書の定義が最優先**です（例: 「当社」）。

| 用語（正規表記） | 定義（要約） | 別表記／表記ゆれ | 注意点 |
|---|---|---|---|
| ウェブコイン | GameID内で利用できる共通ポイントサービス（POKERWEBのオプションサービスとして運用される） | コイン / Webcoin / POKERWEB COIN | 規約文書の定義に従うこと。送付時に追加消費（例: 5%）等のルールは文書で確認 |

## 未確定／確認待ち（追記用）

新規用語や、定義が不明確な表現はここに一時的に置き、確定したら上の表に移してください。

- （例）「加盟クラブ」: 規約上の「登録事業者」との関係整理が必要
