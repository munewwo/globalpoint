# GameID プライバシーポリシー

本プライバシーポリシーは、合同会社グローバルポイント（当社）が提供する「GameID」および当社が提供するウェブコインに関連して、会員（プレイヤー）の利用者情報をどのように取り扱うかを定めるものです。

当社は、当社ウェブサイト上で掲示するプライバシーポリシー（`https://pokerguild.jp/pokerweb/policy/`）およびアプリ内に掲示する内容に基づき、情報の安全性とプライバシーを重視してデータを取り扱います。GameIDの登録・利用に際しては、本プライバシーポリシーの全文をご確認のうえ、同意いただく必要があります。

## 0. 定義

本プライバシーポリシーにおいて使用する以下の用語は、各々以下に定める意味を有するものとします。

1. 「当社」とは、合同会社グローバルポイントをいいます。
2. 「システム提供会社」とは、PW SOFTWARES DE GESTAO LTDAをいいます。
3. 「GameID」とは、システム提供会社が提供するポーカーアプリ「GameID」をいいます。
4. 「ウェブコイン」とは、GameID内で利用できる共通ポイントサービスをいいます。
5. 「会員」とは、GameIDの利用者として登録された個人をいいます。
6. 「登録事業者」とは、当社が別途定める条件に基づいて、ウェブコインを取り扱う事業者をいいます。
7. 「会員情報」とは、会員がGameIDの利用にあたり登録する情報（氏名、メールアドレス、電話番号等）をいいます。
8. 「開示等」とは、開示、訂正、追加、削除、利用停止、消去その他法令上認められる請求をいいます。

## 1. 取得する情報

当社は、GameIDおよびウェブコインの提供にあたり、以下の情報を取得する場合があります（これらに限りません。）。

- 会員情報（氏名、メールアドレス、電話番号等）
- 本人確認に関する情報（eKYCに必要な情報、提出書類、確認結果等）
- アプリ・サービス利用に伴い生成・記録される情報（ログ、端末情報、アクセス情報等）

## 2. 利用目的

当社は、取得した情報を、以下の目的で利用します。

- GameIDおよびウェブコインの提供、運用、保守、改善
- 本人確認（eKYC）、不正利用防止、セキュリティ確保
- 問い合わせ対応、サポート提供
- 法令遵守、紛争対応

## 3. 第三者提供・委託

当社は、法令に基づく場合または本人の同意がある場合等を除き、取得した情報を第三者に提供しません。ただし、当社は、GameIDの運用、本⼈確認その他の業務の全部または一部を、システム提供会社その他の第三者に委託することがあります。また、当社は、会員の同意に基づき、会員情報の一部を登録事業者に共有する場合があります。

## 4. 安全管理措置

当社は、取得した情報の漏えい、滅失又は毀損の防止その他の安全管理のために、合理的な措置を講じます。

## 5. 開示等の請求

当社は、会員から、当社が保有する情報について、開示等を求められた場合、法令の定めに従って適切に対応します。

## 6. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、当社が別途案内する窓口にて受け付けます。

## 7. 改定

当社は、法令対応、運用上の必要その他当社が必要と判断する場合、本プライバシーポリシーを変更することがあります。変更後の内容は、当社ウェブサイト上での掲示その他の方法により周知します。
