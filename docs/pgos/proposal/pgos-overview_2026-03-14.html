1. プロジェクト概要と背景

PGOS は、全国および海外のポーカー店舗・大会運営に必要な機能を統合的に提供するプラットフォームです。現行の POKERWEB（店舗オペレーション＆マネジメント）、GameID（プレイヤー向け会員アプリ）、POKERGUILD（スケジュールサイト）を統合し、自社開発の新システムとして構築します。

項目内容
企画・運営合同会社グローバルポイント
対象ユーザーポーカー店舗・大会主催者（運営側）、ポーカープレイヤー（利用者側）
置き換え対象POKERWEB (247クラブ) / GameID / POKERGUILD (306クラブ)
展開スコープ初期からグローバル対応（多言語・多通貨）

現行サービスの課題

  • POKERWEB / GameID はブラジル開発元への依存度が高く、日本市場向けの機能追加・改善の自由度が低い
  • POKERGUILD は情報掲載サイトにとどまり、店舗オペレーションとの連携が限定的
  • 競合（Poker Fans 450店舗超、Waitinglist 230店舗超）が急速にシェアを伸ばしている

PGOS が目指す 5つの柱

  1. 店舗運営の一元管理 — トーナメント・リングゲーム・会員・売上を一つのシステムで
  2. プレイヤー体験の向上 — 店舗発見〜ウェイティング〜戦績追跡〜ポイント管理をアプリで完結
  3. ポイントエコシステムの進化 — ウェブコインを発展させたコンプライアンス対応済みのポイントシステム
  4. グローバル展開 — 多言語・多通貨対応を初期設計に組み込む
  5. データ活用 — 戦績データベースの構築によるランキング・分析基盤（Hendon Mob / GPI 対抗）

4. ポイントシステム（ウェブコイン後継）

PGOSの最大の差別化要素。競合（Poker Fans / Waitinglist）にはない加盟店共通ポイントエコシステムを構築する。

ポイントのライフサイクル

  1. クラブ → 当社に広告料を支払い → ポイント付与枠（付与可能額）が設定される WC-1-01
  2. プレイヤーがトーナメント入賞 or リングゲームで成果 → クラブが付与リクエストを送信 WC-2-01
  3. 当社が自動バリデーション後に承認 → プレイヤーにポイント付与（選手契約 + eKYC + 18歳以上が前提） WC-2-03〜06
  4. プレイヤーが加盟店でポイント利用 → エントリー費・遊技料・飲食に使える（利用時 5% 手数料） WC-4-01〜04
  5. クラブが当社に精算申請 → 広告料の範囲内で精算される（勘定科目: 両替、消費税なし） WC-6-01〜03

法的設計の原則

原則内容
販売しない対価を受け取らないため、資金決済法上の「前払式支払手段」に該当しない
付与の主体は当社店舗が直接付与することはできない。当社が承認して付与する
景品ではない選手契約（業務委託契約）に基づく報酬として位置づけ
等価禁止遊技チップとポイントを等価にする構造は禁止（システムで自動検証）

システム的に強制する禁止事項

チェック内容実装
登録事業者によるプレイヤーへの直接付与付与フローに当社承認を必須化
遊技チップ＝ポイントの等価構造変換率テーブルで自動チェック
高額サービス（トーナメント税別10万円超等）設定時に自動ブロック
未成年者のポイント利用eKYCの年齢確認と連動

PGOS の差別化ポイント

  1. 全方位型の統合力 — Poker Fans はトーナメント、Waitinglist はリングゲームに強い。PGOS は両方を高水準で統合し、さらにポイントシステムを掛け合わせる
  2. ポイントエコシステム — 加盟店共通のポイントシステムは PGOS 独自の優位性。コンプライアンス対応済みの設計で風営法・資金決済法をクリア
