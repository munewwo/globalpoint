# PGOS 競合分析

## 概要

PGOSの設計にあたり、国内外の主要な競合サービスを分析し、差別化ポイントを明確にする。

---

## 1. 国内競合

### 1.1 Poker Fans / Poker Club（ポーカーファンズ株式会社）

- **サイト**: https://pokerfans.co.jp/
- **導入規模**: 450店舗超（2025年時点）
- **プロダクト構成**: SunVy Poker（初心者向け）/ Fans Poker（プレイヤー向け）/ Poker Club（店舗管理）

#### 店舗管理機能（Poker Club）

| 機能 | 内容 |
|------|------|
| イベント管理 | トーナメント・リングゲーム・初心者教室の登録・管理 |
| 会員管理 | イベント申込者の自動会員登録、QRコードスキャン受付 |
| 受付機能 | 当日参加者管理・料金設定 |
| ストラクチャー管理 | トーナメント構成設定 |
| ランキング管理 | イベント成績表示 |
| 会場管理 | 複数会場の情報登録・更新 |
| 収益レポート | 日次/週次/月次の売上・会員統計 |
| プッシュ配信 | 会員向けアプリ通知 |
| 店舗サイト生成 | クラブ独立サイトの自動生成 |
| シンクロタイマー | iPhone / iPad / Android TV / Fire TV 対応の同期タイマー |

#### 料金プラン

| プラン | 月額 | 会場数 | 会員上限 | タイマー台数 |
|--------|------|--------|---------|-------------|
| ベーシック | 無料 | 2 | 100 | — |
| パーソナル | ¥8,800 | 10 | 1,000 | 3台 |
| ビジネス | ¥22,000 | 50 | 10,000 | 10台 |
| エンタープライズ | ¥55,000 | 無制限 | 無制限 | 無制限 |

#### 強み

- **無料プラン**があり、小規模店舗の導入ハードルが低い
- マルチデバイス同期タイマーの完成度が高い
- オンライン（SunVy Poker）→ライブ（店舗イベント）への動線
- 集客支援（ポータルサイト掲載）

#### 弱み・PGOSでの差別化余地

- ポイントシステム（ウェブコインに相当するもの）が存在しない
- グローバル展開が弱い（日本国内中心）
- プレイヤー戦績データベース・ランキングが限定的
- リングゲーム管理（チップ自動記録等）が弱い

---

### 1.2 Waitinglist（株式会社Waitinglist / DMM.comグループ）

- **サイト**: https://waitinglist.co.jp/
- **導入規模**: 230店舗超（2025年時点）
- **設立**: 2022年
- **親会社**: DMM.com

#### 主要機能

| 機能 | 内容 |
|------|------|
| ウェイティングリスト | リングゲームの順番待ち登録（リモート対応） |
| チェックイン | 加盟店舗でのチェックイン |
| チップ管理 | リングゲームのチップ増減を自動記録 |
| トーナメント管理 | トーナメント作成・管理 |
| ランキング | リングゲーム・トーナメントのスコア即時反映 |
| 戦績機能 | チップ増減、総プレイ時間の自動記録 |
| フレンド機能 | 来店中のプレイヤーをリアルタイム表示 |
| 店舗検索 | 全国ポーカースポットの条件検索 |
| 飲食注文 | アプリ内でフード・ドリンク注文・決済 |
| ポイント・クーポン | 店舗独自ポイント・クーポン管理 |

#### 強み

- **リングゲーム特化**のチップ自動記録が業界で突出
- DMM.com のバックアップによる資金力・開発力
- リアルタイム来店表示などプレイヤー体験が優れている
- 飲食注文・決済のインアプリ対応

#### 弱み・PGOSでの差別化余地

- トーナメント管理機能が Poker Fans / POKERWEB と比較して弱い
- 加盟店共通のポイントシステムがない（店舗独自ポイントのみ）
- グローバル展開が想定されていない
- ウェブコインのような業界横断のエコシステムがない

## 2. 海外競合・参考サービス

### 2.1 Hendon Mob / Global Poker Index (GPI)

- **サイト**: https://pokerdb.thehendonmob.com/
- **データ規模**: プレイヤー 899,000人超 / イベント 654,000件超 / リザルト 5,597,000件超

#### 主要機能

| 機能 | 内容 |
|------|------|
| プレイヤーデータベース | 世界最大のライブポーカー戦績データベース |
| ランキング | All Time Money / 年別 Money / GPI / Player of the Year / FlagHunter 等 |
| 国・地域フィルタ | 国別・地域別ランキング表示 |
| イベントカレンダー | フェスティバル・ツアー・サーキット情報 |
| プレイヤーフォロー | プロフィール更新・リザルト追加の通知（メール） |
| 会場データベース | ポーカー会場情報 |

#### PGOSへの示唆

- プレイヤー戦績データベースは PGOS の強力な差別化要素になり得る
- 日本（およびアジア）のライブポーカー戦績は Hendon Mob でカバーが薄い
- PGOS で蓄積した戦績データを API で提供すれば、Hendon Mob との連携も可能

---

### 2.2 Poker TDA / トーナメント管理ソフトウェア

- **Poker TDA** (Tournament Directors Association): トーナメントルールの標準化団体
- **関連ソフト**: Poker Tournament Manager、The Tournament Director (TDT) 等

#### PGOSへの示唆

- Poker TDA ルールへの準拠はプロフェッショナルなトーナメント運営の必須要件
- 海外の大規模トーナメントで使用されているソフトの機能（シート抽選、テーブルバランス、ペイアウト計算）は PGOS でも対応が必要
- ストラクチャーテンプレートの共有・インポート/エクスポート機能が差別化になる

---

### 2.3 Blind Timer / Poker Timer 系アプリ

各種無料・有料のポーカータイマーアプリ（Poker Timer Pro、Blinds Timer 等）。

#### PGOSへの示唆

- 多くの小規模店舗はこれらの無料アプリで代用している
- PGOS のタイマー/クロック機能がこれらより使いやすければ、導入の動機になる
- マルチデバイス同期、カスタムブランディング、リアルタイム参加者表示などで差別化

---

## 3. 現行サービス（POKERWEB / POKERGUILD）

### POKERWEB

- **導入規模**: 247クラブ（ウェブコイン利用 180クラブ）
- **開発元**: PW SOFTWARES DE GESTAO LTDA（ブラジル）
- **料金**: ¥5,500〜¥27,900/月（Bronze〜Diamond）+ セットアップ ¥12,000

#### 主な機能

トーナメント管理 / リングゲーム管理 / 顧客管理 / ウェブコイン / ランキング / 予約管理 / テーブル調整 / 飲食会計 / 権限管理 / レポート

#### PGOSで引き継ぐべき要素

- **ウェブコイン**のエコシステム（付与・利用・精算フロー）
- **5段階の料金プラン**による規模別対応の考え方
- **AWS ベースのインフラ**の信頼性
- 既存247クラブの契約・データ

### POKERGUILD

- **導入規模**: 306クラブ
- **機能**: イベントスケジュール掲載サイト、選手契約管理

#### PGOSで引き継ぐべき要素

- **スケジュールサイト**としてのブランド認知
- **選手契約管理**のワークフロー
- 広告料ベースのビジネスモデル

## 5. PGOS の差別化戦略（まとめ）

### 5.1 他社にない統合力

Poker Fans はトーナメント管理に強く、Waitinglist はリングゲームに強い。PGOS はこの両方を高水準で統合し、さらにポイントシステム（ウェブコイン後継）を掛け合わせることで、どの競合にもない「全方位型プラットフォーム」を実現する。

### 5.2 ポイントエコシステム

加盟店共通で使えるポイントシステムは PGOS 独自の優位性。コンプライアンス対応済みの設計により、風営法・資金決済法の要件を満たしつつ、プレイヤーのエンゲージメントと店舗の集客力を高める。

### 5.3 戦績データベース

Hendon Mob がカバーしきれないアジア圏のライブポーカー戦績を網羅するデータベースを構築。プレイヤーにとっての「実績の証」となり、アプリの継続利用を促進する。

### 5.4 グローバルファースト設計

国内競合（Poker Fans / Waitinglist）は日本市場に特化している。PGOS は多言語・多通貨対応を初期設計に組み込み、既存の POKERWEB がカバーしていた海外市場も取り込む。

### 5.5 無料プランによる導入障壁の低減

Poker Fans の無料ベーシックプランは450店舗超の導入を牽引した。PGOS でも基本機能を無料で提供し、ポイントシステムや高度な分析機能を有料化するフリーミアムモデルを検討する。
