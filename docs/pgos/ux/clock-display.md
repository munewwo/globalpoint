# PGOS Clock — トーナメントタイマー UX設計

## 対象

- **アプリ**: PGOS Clock（スマートTV / Fire TV / ブラウザ向けフルスクリーン表示）
- **ユーザー**: 店舗オペレーター（表示用）、プレイヤー（閲覧用）
- **フェーズ**: Phase 1 (MVP)
- **操作方法**: PGOS Clock 自体は「表示専用」。操作はすべて PGOS Manager (M-TN-LIVE) から行い、WebSocket で同期

---

## 設計原則（Clock固有）

- **遠くからでも読める**: 最低5m離れた位置から主要情報が読めるフォントサイズ
- **暗い環境に最適化**: ポーカールームの照明（暗め）を前提としたハイコントラスト配色
- **ダークベース**: 黒背景 + 明るい文字が基本。ブランドカラーはアクセントのみ
- **最小限の情報**: 一画面に表示する情報は厳選し、視線の移動を最小化
- **カスタムブランディング**: 店舗ロゴ・配色をカスタマイズ可能

---

## 画面一覧

| 画面ID | 画面名 | 用途 |
|--------|--------|------|
| C-MAIN | メインクロック | トーナメント進行中の主表示 |
| C-BREAK | ブレイク | 休憩時間の表示 |
| C-INFO | 情報オーバーレイ | ペイアウト・参加者情報の表示 |
| C-STANDBY | 待機画面 | トーナメント開始前/終了後の告知 |

### C-MAIN — メインクロック

| 項目 | 内容 |
|------|------|
| **画面ID** | C-MAIN |
| **要件ID** | B1-2-01〜B1-2-09 |
| **目的** | トーナメント進行中の主要情報を大画面で表示 |

```
├─────────────────────────────────────────────────────┤
│                                                     │

│              400 / 800  Ante 100                    │
│                                                     │
│                                                     │
│                   12 : 34                           │
│                                                     │
│                                                     │
│  ┌──────────────┐                 ┌──────────────┐  │
│  │ 次のレベル     │                 │ 残りプレイヤー  │  │
│  │ 500 / 1000   │                 │   14 名       │  │
│  │ Ante 200     │                 │              │  │
│  └──────────────┘                 └──────────────┘  │
│                                                     │
│  次のブレイク: Lv6 終了後                              │
│                                                     │
├──────────┬──────────┬──────────┬────────────────────┤

│    24    │    8     │    2    │    28,500         │
└──────────┴──────────┴──────────┴────────────────────┘
```

**レイアウトのポイント:**
- レベル番号: 画面上部中央、大きめフォント
- ブラインド/アンティ: 最も目立つ最大フォント
- 残り時間: カウントダウン表示、最も大きなフォント
- 次のレベル情報: 左下に控えめに表示
- 残りプレイヤー: 右下に表示
- フッター: 統計情報（エントリー数、リバイ数、平均スタック）

**カウントダウンの視覚フィードバック:**
- 残り5分以下 → 時間表示が黄色に変化
- 残り1分以下 → 時間表示が赤に変化 + 点滅
- レベル終了 → サウンドアラート + 次レベル自動切替

### C-STANDBY — 待機画面

| 項目 | 内容 |
|------|------|
| **画面ID** | C-STANDBY |
| **要件ID** | B1-2-06 |
| **目的** | トーナメント開始前や終了後に次のイベント情報を表示 |

```
┌─────────────────────────────────────────────────────┐
│  [店舗ロゴ]                                  [PGOS]  │
├─────────────────────────────────────────────────────┤
│                                                     │
│                                                     │
│                   [店舗名]                           │
│                                                     │
│                                                     │
│               NEXT TOURNAMENT                       │
│                                                     │
│           🏆 デイリーNLH                              │
│                                                     │
│              19:00 START                            │
│                                                     │
│          Entry: ¥3,000                              │
│          Re-buy: ¥2,000                             │
│                                                     │
│          エントリー受付中: 12名                        │
│                                                     │
│                                                     │
│                                                     │
├─────────────────────────────────────────────────────┤
│  [店舗ロゴ]       本日のスケジュール         [QRコード] │
│  19:00 デイリーNLH  |  21:00 ディープスタック          │
└─────────────────────────────────────────────────────┘
```

**表示内容の自動切替:**
- 次のトーナメントがある場合 → 次のトーナメント情報
- トーナメント終了直後 → 結果サマリー（上位入賞者）を一定時間表示
- スケジュールがない場合 → 店舗ロゴ + スケジュール一覧

**QRコード:**
- フッター右にQRコードを表示し、プレイヤーが PGOS Player アプリでスキャンしてエントリー可能

## デバイス対応

| デバイス | 解像度 | 対応方法 |
|---------|--------|---------|
| スマートTV (16:9) | 1920x1080 | ネイティブ対応 |
| Fire TV | 1920x1080 | Fire TV アプリ or ブラウザ |
| Android TV | 1920x1080 | Android TV アプリ or ブラウザ |
| PCモニター (ブラウザ) | 任意 | フルスクリーン表示対応 |
| タブレット (横置き) | 任意 | タッチ操作は不要（表示専用） |

## 同期仕様

- PGOS Manager (M-TN-LIVE) で行った操作が即座に反映
- WebSocket による双方向通信
- ネットワーク切断時はローカルでタイマー継続（再接続時に同期）
- 同期ずれは2秒以内を保証
