# PGOS Portal — 公開Webサイト UX設計

## 対象

- **アプリ**: PGOS Portal（公開Webサイト、SSR/SSG）
- **ユーザー**: 一般公開（ログイン不要で閲覧可能）、ポーカープレイヤー、検索エンジン
- **フェーズ**: Phase 1 (MVP) + Phase 2 (拡張)
- **目的**: 店舗・イベント情報の公開、ランキング・戦績の公開、SEO対策、PGOS Player アプリへの誘導

---

## 設計原則（Portal固有）

- **SEO ファースト**: サーバーサイドレンダリング (SSR) または静的生成 (SSG) で検索エンジンにインデックスされやすい構造
- **構造化データ**: Schema.org のマークアップで店舗情報・イベント情報をリッチスニペット対応
- **アプリ誘導**: 各ページにアプリダウンロードへの導線を設置
- **レスポンシブ**: モバイル / タブレット / デスクトップに対応
- **高速表示**: LCP (Largest Contentful Paint) 2.5秒以内
